# NotesApp


## Features

- Get Notes 
- Create Notes 
- Delete Notes 
- Update Notes 
- Login and register user
- User Roles




-

## Installation


Install the dependencies and devDependencies and start the server.

```sh
cd NotesAoo
npm i
node run start
```

For production environments...

```sh
npm install --production
NODE_ENV=production node run start
```
